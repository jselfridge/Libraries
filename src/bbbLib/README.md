
bbbLib
======

This directory is C code for a static library that delivers linear 
algebra functionality.  The functions are grouped in the following 
categories:

      bbbLED      - 
      bbbGPIO     - 
      bbbUART     - 
      bbbI2C      -
      bbbMPU      - 
      bbbPRU      - 
      bbbServo    -
      bbbRadio    -
